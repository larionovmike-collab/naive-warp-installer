# 🚀 VPN Installer (Caddy + NaiveProxy)

Простой установщик VPN-прокси сервера с автоматической настройкой.

---

## 📌 Что это такое?

Этот скрипт автоматически разворачивает:
- ⚡ VPN прокси сервер (Caddy + forward proxy)
- 🔐 Авторизацию (логин/пароль)
- 🌐 Работу через домен + SSL
- 🎭 Маскировку под обычный сайт
- ⚙️ Автозапуск через systemd

---

## 💻 Требования

Перед установкой:
- Ubuntu 20.04 / 22.04 / 24.04 И Debian 11 / 12 / 13
- Root доступ (sudo)
- Привязанный домен (A-запись на сервер)

---

## 📦 Установка

Скопируй и выполни:

```sh
bash <(curl -sSL https://raw.githubusercontent.com/larionovmike-collab/naive-warp-installer/refs/heads/main/install.sh)
```

---

## ⚙️ Что будет дальше?

Скрипт задаст вопросы:

- 🌐 Домен
- 📧 Email (для SSL)
- 🎭 Фейковый сайт (маскировка)
- 👤 Логин (или авто)
- 🔑 Пароль (или авто)
- ⚡ Установка WARP (да/нет)

---

## 🎯 После установки ты получишь:

- Адрес сервера: https://твой-домен
- Логин и пароль для подключения
- Готовый SOCKS/HTTPS proxy конфиг

---

## 📱 Пример клиента

```json
{
  "listen": "socks://127.0.0.1:20808",
  "proxy": "https://login:password@domain.com"
}
```
